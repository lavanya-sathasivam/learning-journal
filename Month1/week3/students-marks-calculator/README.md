# Students Marks Calculator

    Students Marks Calculator is a Python console application that stores student information, calculates average mark, assigns a grade and displays a formatted report

# Features
- Stores sudent details
- Stores marks using lists
- Stores student records using dictionaries
- Calculates average mark
- Generate grade
- Display formatted report

# How to run
    python student_marks.py

# Sample usage

Enter student name : Lavanya
Enter roll no : 030
Enter marks for Python : 95
Enter marks for Java : 80
Enter marks for C : 70
Enter marks for Maths : 99


STUDENT REPORT
Name        : Lavanya    
Roll No     : 030
Marks       : [95,80,70,99]
Average     : 86.00
Grade       : A
