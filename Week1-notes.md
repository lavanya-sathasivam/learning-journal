# Git and GitHub Fundamentals

Git
- It is a version control system
- It is used to keep track of code changes

GitHub
- It is web based git hosting service
- Learned various git commands and their uses

Commands
- clone : cloning existing repository to code editor
- status : to track file's status in github and local system 
- add : adds files / changes in a file to staging area 
- commit : transfers files from staging area to local git folder
- push : uploads the files from local git folder to the github's folder
- pull : pull files or file changes from the github folder to the local system folder
- init : initializes empty git repository 
- branch : checks which branch are we working in
- checkout : changes branch 

- Learned about how pull requests work
- Learned about creating different branches
- Learned about conflicting issues
- Practiced git commands by creating a repository and commiting the changes
